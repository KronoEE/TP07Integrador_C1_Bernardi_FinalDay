# TP07Integrador_C1_Bernardi_FinalDay

Final Day es un juego de supervivencia 2D creado en Unity, inspirado en la tensión y el estilo de Project Zomboid.
Controlás a un sobreviviente en un mundo invadido por zombies, donde cada decisión puede ser la diferencia entre vivir o morir.

Tu objetivo es sobrevivir el mayor tiempo posible, enfrentar hordas de enemigos y manejar el estado emocional del personaje mientras luchás contra la amenaza constante.

Explorá, dispará, controlá tus emociones y resistí hasta el último día.

## 🕹️ Controles

| Acción | Tecla / Botón |
|--------|----------------|
| Mover a la izquierda | `A` |
| Mover a la derecha | `D` |
| Saltar | `Espacio` |
| Disparar | `E` |
| Correr | `Left Shift` |

⚔️ Mecánicas Principales

🧟 Zombies: enemigos que persiguen y atacan al jugador constantemente.

🔫 Sistema de combate: el personaje utiliza una pistola para defenderse.

❤️ Sistema emocional: el personaje reacciona al peligro.

Aumenta el estrés al ser atacado.

Siente miedo al ver muchos zombies.

Sus emociones afectan el gameplay.

🌍 Supervivencia: gestioná el peligro constante y mantenete con vida el mayor tiempo posible.
